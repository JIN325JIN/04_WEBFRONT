text 관련된 input 태그

type = "text" 

    -한줄 짜리 문자열(텍스트) 입력 할 수있는 입력 상자
    - input 태그의 type의 기본값으로, 생략가능
    - inline 형식 (모든 input 태그가 inline)
  

    
text관련 input 태그의 공용 속성

    -size : 입력상자 크기
    -maxlength :입력 받는 텍스트의 최대 길이
    - place holder : 입력 상자에 작성될 내용을 설명

form태그

  - div와 같은 영역(block 형식)
  - 내부에 작성된 input 태그의 값을 서버로 전달하는 역할

  [속성]
  1) action : 내부에 작성된 값을 전달할 서버 또는 페이지 주소를 작성
   (미작성 시 기본값으로 현재 페이지의 URL 적용)
  2) name : form 태그의 이름을 지정하여 form 태그 구분(JS 사용)
  3) method(방식) : 서버로 데이터 전달하는 방식을 지정하는 속성
    - GET / POST 지정 가능(기본값 GET)
  4) target : action에 지정된 주소를 현재탭/새탭 중 어디서 열지 지정
